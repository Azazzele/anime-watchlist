# anime-watchlist

Anime Watchlist представляет собой современное веб-приложение для отслеживания просмотренных аниме-сериалов, разработанное с использованием передовых технологий веб-разработки.

Технологическая основа приложения построена на комбинации мощного фронтенд-фреймворка Vue 3 и производительного бэкенд-фреймворка FastAPI. Такой стек технологий обеспечивает высокую скорость работы и удобство использования.

Функционал приложения включает в себя возможность поиска аниме по названию, добавления их в персональный список, отслеживания количества просмотренных серий и сохранения прогресса просмотра. Система позволяет пользователям эффективно управлять своим списком аниме.

Пользовательский интерфейс разработан с учетом современных тенденций веб-дизайна, обеспечивая интуитивно понятное взаимодействие. Приложение адаптируется под различные устройства и размеры экрана, что делает его удобным для использования как на компьютерах, так и на мобильных устройствах.

Архитектура проекта организована с разделением на фронтенд и бэкенд компоненты, что обеспечивает четкое разделение ответственности и упрощает поддержку кода. Использование Vue 3 позволяет создавать реактивные компоненты, а FastAPI обеспечивает быструю обработку запросов и взаимодействие с базой данных.
